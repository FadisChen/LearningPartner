# iPAS AI 應用規劃師 - 智慧學習中心 Learning Hub

這是一個專為「iPAS AI 應用規劃師」考照準備的互動式學習平台。整合了知識架構圖 (Mind Map) 與 Google Gemini AI，讓您可以像使用 AI 家教一樣，針對官方學習指引進行問答與重點解析。

## ✨ 主要功能

- **🚀 知識地圖可視化**：
  - 完整收錄三個科目的考試重點。
  - 使用 Markmap 技術呈現動態心智圖。
  - **點擊互動**：點擊節點即可叫出 AI 選單（解釋概念、產生考題）。

- **📝 自動生成心智圖**：
  - **上傳即生成**：點擊左側「新增科目」，上傳任意 PDF 文件（如技術論文、筆記）。
  - **AI 摘要**：Gemini 會自動分析文件內容，生成結構化的知識地圖。
  - **動態擴充**：生成的主題會自動加入科目選單，立即可以進行點擊互動學習。
  - **自動快取**：上傳的 PDF 會自動儲存在瀏覽器中，切換科目時無需重新上傳。

- **🤖 AI 智能家教 (Gemini)**：
  - **自帶 Key (BYOK)**：使用您個人的 Google Gemini API Key，確保隱私與控制權。
  - **多文件支援**：每個科目可上傳多個 PDF 文件，AI 會綜合所有文件回答問題。
  - **文件對話 (RAG-like)**：直接上傳官方 PDF 學習指引，AI 會基於文件內容回答問題。
  - **引用來源追蹤**：AI 回答時會標註來源章節，方便對照原文。
  - **多輪對話**：AI 會記住同一科目內的對話歷史，支援連續追問與深入討論。
  - **獨立對話記錄**：每個科目擁有獨立的對話記錄，切換科目時會自動載入該科目的對話歷史。
  - **流式輸出**：回應即時顯示，無需漫長等待。
  - **圖文整合**：在對話中支援 Markdown 格式渲染。

- **📚 學習工具**：
  - **生成學習指南**：一鍵生成完整的學習指南，包含重點摘要、FAQ、考試重點、建議學習順序。
  - **建議問題清單**：自動生成 10 個建議問題，涵蓋基礎、進階、應用三種難度。

- **💾 智能快取系統**：
  - **多檔案快取 (IndexedDB)**：上傳的 PDF 會自動儲存在瀏覽器本地資料庫中，支援多檔案。
  - **自動載入**：切換回已上傳 PDF 的科目時，系統會自動載入所有檔案，無需重新上傳。
  - **檔案管理**：可以查看已上傳的檔案列表，並隨時刪除不需要的檔案。
  - **對話保存**：每個科目的對話記錄會保留在記憶體中（重新整理頁面後會清除）。
  - **智能提示**：針對預設科目，系統會提示建議上傳的檔案名稱。

- **⚙️ 高度客製化**：
  - **模型選擇**：預設使用 `gemini-2.5-flash`，或自訂其他模型 ID。
  - **繁體中文介面**：全在地化設計，親切易用。

## 🛠️ 使用方式

1. **啟動應用**：
   直接使用瀏覽器（推薦 Chrome 或 Edge）開啟 `LearningHub.html` 檔案。

2. **設定 API Key**：
   - 首次使用請點擊左上角的輸入框。
   - 貼上您的 [Google Gemini API Key](https://aistudio.google.com/app/apikey)。
   - 點擊儲存（Key 僅儲存於您的瀏覽器 LocalStorage，不會上傳至任何第三方伺服器）。

3. **開始學習**：
   - **切換科目**：點擊左側 1, 2, 3 按鈕切換預設科目。
   - **新增科目**：點擊「＋ 新增科目」按鈕，上傳 PDF，AI 將為您生成專屬的心智圖。
   - **文件問答**：針對當前科目，點擊下方虛線框上傳 PDF，讓 AI 根據文件回答問題。
     - **首次上傳**：系統會將檔案快取在瀏覽器中。
     - **自動載入**：之後切換回該科目時，會自動載入快取的檔案。
   - **互動**：
     - 在地圖上點擊節點 -> 選擇「💡 解釋」或「❓ 出題」。
     - 切換到「AI 導師」分頁 -> 自由提問。
   - **多輪對話**：
     - 在同一科目內，AI 會記住之前的對話內容。
     - 切換科目時，對話記錄會自動切換到該科目的歷史記錄。

## ⚠️ 注意事項

- 本工具為純前端應用 (Client-Side App)，不會儲存您的對話紀錄至伺服器。
- **檔案快取**：上傳的 PDF 會儲存在瀏覽器的 IndexedDB 中，僅限於當前瀏覽器使用。
  - 若清除瀏覽器資料，快取的檔案會一併清除。
  - 不同瀏覽器或裝置之間不會同步。
- **對話記錄**：每個科目的對話記錄保存在記憶體中，重新整理頁面後會清除。
- PDF 解析使用瀏覽器內建 API 轉換為 Base64 傳送給 Gemini，請留意 API 使用量限制。
- 若遇到 API Error，通常是 API Key 無效或配額已滿。
- AI 生成的科目與心智圖僅存於**當前工作階段**，重新整理頁面後會消失（預設科目則會保留）。
